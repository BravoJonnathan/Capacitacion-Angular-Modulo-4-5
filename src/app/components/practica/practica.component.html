Práctica realizada

Para la practica se realizo lo siguiente:

  1. Como pasos iniciales se procedio a crear el nuevo proyecto e instalar la dependencia de boostrap.

  2. Luego se continuo con la creación de los 5 componentes para las opciones descritas en la tarea

  3. Una vez creado los componentes de angular, se procedio a configurar las rutas. Para ello se siguieron los siguientes pasos:

    1. Se importaron los componentes anteriormente creados en app-routing.module.ts.
    2. Se definieron las rutas y a los componente que llamarian dichas rutas dentro de la constante routes

  4. Una vez configuradas las rutas, se creo el menu navbar en app.component.html y con router-link se indico la ruta a cada opcion del menu.

  5. Luego se procedio a agregar el contenido indicado en la descripcion de la tarea a cada componente.
