Data Binding

A breve rasgos data binding es la manera en que se comunican las variables del código typescript con su template HTML.


Template expression

Este es el enlace unidireccional de la propiedad de un componente a la plantilla del componente. Entonces, cuando cambia la propiedad en los componentes, la plantilla se actualiza para reflejar los nuevos cambios.

Property binding

Este es un enlace unidireccional de los datos de un modelo al destino de enlace. El destino de enlace puede ser propiedades de elementos, propiedades de componentes o propiedades de directivas directivas. Entre los corchetes se []encuentra el nombre de la propiedad del modelo que se vinculará al elemento.

Event Binding

En el DOM, los elementos pueden generar eventos y vinculamos cada evento con un controlador de eventos, una función que se ejecuta cuando se emite el evento.
